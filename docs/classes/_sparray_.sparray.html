Class Sparray<T>

Sparray - Supper Array Holds the methods and attributes of a sparray. Sparrays are immutable, i.e. it is not possible to include, remove or replace an element of a sparray (although it is possible to change the referenced element, e.g: attributes of an object). Every operation/transformation (such as map, filter, and concat) will generate a new and immutable sparray.
